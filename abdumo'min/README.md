# Abdumo'min

## Sizning komponentlaringiz

| # | Komponent | Fayl yo'li (qayerda kod yozasiz) |
|---|-----------|----------------------------------|
| 1 | `ImageUploadZone.jsx` | `abdumo'min/ImageUploadZone.jsx` |
| 2 | `ContactForm.jsx` | `abdumo'min/ContactForm.jsx` |
| 3 | `PricingAndSubmitForm.jsx` | `abdumo'min/PricingAndSubmitForm.jsx` |

### Qanday kod yozish kerak?
- Faylni oching: `abdumo'min/ImageUploadZone.jsx` (yoki boshqa komponent)
- Ichidagi JSX kodni Tailwind CSS klasslari bilan yozing
- `useState`, `useEffect` kabi React hooklaridan foydalanish MUMKIN
- Faqat UI dizaynni chiroyli qiling

## Sizning sahifalaringiz

Bu sahifalarga kirib o'z komponentlaringizni ko'rish uchun:

| # | Sahifa | Route (brauzerga yozing) |
|---|--------|--------------------------|
| 1 | Trade-In | http://localhost:5173/trade-in |
| 2 | Medical Workers | http://localhost:5173/medical-workers |
| 3 | Recycling | http://localhost:5173/recycling |
| 4 | Credit Installment | http://localhost:5173/credit-installment |

## Yo'riqnoma
1. `npm run dev` terminalda ishga tushiring
2. Brauzerda http://localhost:5173/trade-in ni oching
3. O'z komponentlaringizni `abdumo'min/` papkasidagi fayllarda tahrirlang
4. O'zgarishlarni ko'rish uchun sahifani qayta yuklang
